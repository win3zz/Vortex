## How to Edit 

1. Open this project in **Google Cloud Shell** using the button below: 

   [![Open in Cloud Shell](https://gstatic.com/cloudssh/images/open-btn.svg)](https://shell.cloud.google.com/cloudshell/editor?cloudshell_git_repo=https%3A%2F%2Fgithub.com%2Fwin3zz%2FVortex&cloudshell_workspace=%2F)

2. Edit files using the built-in editor. Here is what you can customize:

   * **Text:** Change the text in `js/vortex.js` inside `drawText()` function.
   * **Colors:** Update `ctx.fillStyle` and `ctx.shadowColor` in `vortex.js`.
   * **Particles:** Adjust the `COUNT` and `speed` values in `js/vortex.js` to change particle numbers or animation speed.

3. Save your changes and refresh the [preview](https://docs.cloud.google.com/shell/docs/using-web-preview) to see the updates instantly!
